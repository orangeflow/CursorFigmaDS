概览 Overview

WeChat Pay HK 设计系统是一个完整的 CSS 设计系统规范，基于 Figma 设计系统生成。本系统提供了完整的组件样式、设计令牌和工具类，帮助开发者快速构建符合设计规范的界面。

快速开始

<link rel="stylesheet" href="design-system.css">

文件结构

  • design-system.css - 设计系统 CSS 主文件
  • design-system-examples.html - 组件使用示例
  • components-documentation.html - 组件文档（本文件）
  • README.md - 使用文档

组件结构

Picker 组件由以下部分组成：

  • .picker__header - 头部区域（包含关闭按钮和标题）
  • .picker__content - 内容区域（可滚动的选项列表）
  • .picker__columns - 列容器
  • .picker__column - 单列容器
  • .picker__option - 选项项
  • .picker__mask-top - 顶部渐变遮罩
  • .picker__mask-bottom - 底部渐变遮罩
  • .picker__footer - 底部操作区（包含确定按钮）

选项状态

<!-- 未选中状态 -->

  <span class="picker__option-text">选项文本</span>
</div>

<!-- 选中状态 -->
<div class="picker__option picker__option--selected">
  <span class="picker__option-text">选项文本</span>
</div>

动画参数

Picker 组件的滚动动画参数（需要 JavaScript 实现）：

  • 停止动画时间: 0.3秒 - 当滚动停止时的回弹动画持续时间
  • 惯性时间: 150ms - 影响滚动的"灵敏度"和惯性距离
  • 时间窗口: 100ms - 用于计算惯性滚动的时间窗口
  • 最小滑动距离: 10px - 小于此值视为点击而非滑动
  • 惯性判断阈值: 100ms - 超过此时间间隔则不执行惯性滚动
  • 采样点数量: 最多记录40个滑动轨迹点

步骤器 Step

用于任务步骤展示或任务进度展示。支持水平（Horizontal）和垂直（Vertical）两种布局方式。

基础用法

<!-- 水平步骤器 -->

    <div class="step__icon">

        <span class="step__icon-number-text">1</span>
      </div>
    </div>
    <div class="step__line step__line--brand"></div>
  </div>
  <div class="step__item step__item--in-progress">
    <div class="step__icon">
      <div class="step__icon-number step__icon-number--green">
        <span class="step__icon-number-text">2</span>
      </div>
    </div>
    <div class="step__line"></div>
  </div>
</div>

<!-- 垂直步骤器 -->
<div class="step step--vertical step--process">
  <div class="step__item step__item--completed">
    <div class="step__icon">
      <div class="step__icon-done">
        <svg>...</svg>
      </div>
      <div class="step__line step__line--brand"></div>
    </div>

      <div class="step__header">
        <p class="step__title">标题</p>

      </div>
      <p class="step__summary">摘要信息</p>
    </div>
  </div>
</div>

类型 (Type)

Step 组件支持两种类型：

  • 流程步骤器 (Process Step)：用于引导用户完成多步骤、有先后顺序任务的导航指示组件。它明确展示流程总步骤、当前所处阶段以及已完成和未完成的步骤，强调任务的连续性、顺序性和进度感。
  • 状态指示器 (Status Indicator)：一种用于标记内容区块、页面或独立选项当前所处位置的视觉指示组件。它仅反映当前位置，各"步骤"代表的内容彼此独立，无逻辑先后关系，主要用于空间定位而非进度追踪。
<!-- 流程步骤器 -->
<div class="step step--process">...</div>

<!-- 状态指示器 -->
<div class="step step--indicator">...</div>

布局方向 (Layout Direction)

Step 组件支持水平和垂直两种布局方向。

<!-- 水平布局 -->
<div class="step step--horizontal">...</div>

<!-- 垂直布局 -->
<div class="step step--vertical">...</div>

状态 (Status)

每个步骤项支持三种状态：

  • 已完成 (Completed)：步骤已完成，使用绿色图标和连接线
  • 进行中 (In Progress)：当前正在进行的步骤，使用绿色图标
  • 未开始 (Pending)：尚未开始的步骤，使用灰色图标和连接线
<div class="step__item step__item--completed">已完成</div>
<div class="step__item step__item--in-progress">进行中</div>
<div class="step__item step__item--pending">未开始</div>

图标类型 (Icon Types)

Step 组件支持多种图标类型：

  • 数字图标：显示步骤序号，支持绿色（已完成/进行中）和灰色（未开始）两种颜色
  • 完成图标：对勾图标，表示步骤已完成
  • 时间图标：蓝色时间图标
  • 报错图标：红色错误图标
  • 信息图标：黄色信息图标
  • 更多图标：黑色更多图标

</div>

<!-- 时间图标 -->

连接线 (Connector Line)

  • 无线：最后一个步骤项不显示连接线

内容区域 (Content Area)

垂直布局的 Step 组件支持显示内容区域，包括：

  • 标题：步骤的标题文本
  • 时间：步骤的时间信息（可选）
  • 摘要：步骤的摘要信息（可选）
<div class="step__content">
  <div class="step__header">
    <p class="step__title">标题</p>
    <p class="step__time">2020-4-15 17:23</p>
  </div>
  <p class="step__summary">摘要信息摘要信息摘要信</p>
</div>

组件属性

类名 说明 类型
.step--horizontal 水平布局 布局方向
.step--vertical 垂直布局 布局方向
.step--process 流程步骤器 类型
.step--indicator 状态指示器 类型
.step__item--completed 已完成状态 状态
.step__item--in-progress 进行中状态 状态
.step__item--pending 未开始状态 状态
.step__icon-number--green 绿色数字图标 图标类型
.step__icon-number--gray 灰色数字图标 图标类型
.step__line--brand 绿色连接线 连接线样式
.step__line--hidden 隐藏连接线 连接线样式

图标库 Icon Library

图标作为 UI 构成中重要的元素，一定程度上影响 UI 界面整体呈现出的风格。

基础用法

<span class="icon icon--primary">
  <svg width="48" height="48" viewBox="0 0 24 24" fill="currentColor">
    <path d="..."></path>
  </svg>
</span>

尺寸变体

图标支持三种尺寸：小 (24px)、中 (48px)、大 (64px)。

<span class="icon icon--small icon--primary">...</span>
<span class="icon icon--medium icon--primary">...</span>
<span class="icon icon--large icon--primary">...</span>

属性 (Props)

类名 说明 尺寸
.icon--small 小尺寸图标 24px × 24px
.icon--medium 中等尺寸图标（默认） 48px × 48px
.icon--large 大尺寸图标 64px × 64px

颜色变体

图标支持多种颜色变体，通过 CSS 类控制。

<span class="icon icon--primary">...</span>
<span class="icon icon--brand">...</span>
<span class="icon icon--danger">...</span>
<span class="icon icon--disabled">...</span>

属性 (Props)

类名 说明 颜色
.icon--primary 主色图标 黑色 (#000000)
.icon--brand 品牌色图标 绿色 (#07c160)
.icon--danger 危险色图标 红色 (#fa5151)
.icon--disabled 禁用状态图标 半透明灰色

样式变体

图标支持 Fill（填充）和 Outline（描边）两种样式。

<!-- Fill 样式（默认）-->
<span class="icon icon--primary">
  <svg fill="currentColor">...</svg>
</span>

<!-- Outline 样式 -->
<span class="icon icon--primary icon--outline">
  <svg stroke="currentColor" fill="none">...</svg>
</span>

图标按钮

图标可以作为按钮使用，支持点击交互。

<button class="icon-btn">
  <span class="icon icon--primary">
    <svg>...</svg>
  </span>
</button>

<button class="icon-btn icon-btn--small">...</button>
<button class="icon-btn icon-btn--large">...</button>

属性 (Props)

类名 说明 尺寸
.icon-btn 图标按钮（默认） 48px × 48px
.icon-btn--small 小尺寸图标按钮 32px × 32px
.icon-btn--large 大尺寸图标按钮 64px × 64px

图标与文本组合

图标可以与文本组合使用，提供更好的视觉层次。

<div class="icon-with-text">
  <span class="icon icon--primary">
    <svg>...</svg>
  </span>
  <span>文本内容</span>
</div>

图标容器

使用图标容器可以更好地控制图标的对齐和布局。

<div class="icon-container">
  <span class="icon icon--primary">
    <svg>...</svg>
  </span>
</div>
